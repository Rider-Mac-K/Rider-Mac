## Overview

Rider Professional Suite is a complete .NET development platform with database tools, Docker support, and profiling for professional .NET workflows.

DotTrace and dotMemory profiling analyze application performance and memory usage directly from the IDE, identifying bottlenecks and memory leaks.

ASP.NET and Blazor tools provide project templates, debugging, and deployment tools for web application development.

NuGet package management handles dependency resolution, version updates, and vulnerability scanning.

Rider Professional Suite is the .NET development platform for professional teams that need comprehensive C# and .NET tooling with game development, web, and cloud capabilities.

## Performance & Integration

Rider is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Rider processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows

- Apple Silicon or Intel — Universal Binary
